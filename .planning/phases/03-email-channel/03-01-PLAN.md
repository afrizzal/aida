      can run rehype-sanitize on inbound HTML. Ships its own types.

    Do NOT add Redis, GreenMail, or any other runtime service — IMAP/SMTP are outbound TCP connections
    the worker makes to the operator's own mail server (CLAUDE.md single-server rule).
