- `export async function fetchTicketList(filters, ctx: { db; orgId; userId })`: build a Prisma `where`:
  - view: unassigned → `assigneeId: null`; mine → `assigneeId: ctx.userId`; all → {}.
  - statuses → `status: { in: statuses }`.
  - tagId → `tags: { some: { tagId } }`.
  - cf → `customFieldValues: { some: { customFieldDefinitionId: cf.definitionId, OR: [{ valueText: cf.value }, ...] } }` (match against valueText primarily; numbers/dates best-effort).
  - q present → `const rows = await searchTickets(ctx.orgId, q); const ids = rows.map(r=>r.id); where.id = { in: ids };` (empty q → skip).
  - `return ctx.db.ticket.findMany({ where, orderBy: { createdAt: "desc" }, take: filters.limit ?? 50, include: { contact: true, assignee: true, tags: { include: { tag: true } } } });`
